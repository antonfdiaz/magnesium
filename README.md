# Magnesium
Magnesium is a web browser made in Python with PySide6.

## How to run:
1. Download the file
2. Unzip the file
3. Install PySide6 with pip (```pip3 install pyside6```)
4. Run the script with Python (```python3 main.py```)
5. Done!
